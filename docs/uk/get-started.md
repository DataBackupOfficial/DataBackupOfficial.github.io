# Розпочнемо

[DataBackup](https://github.com/XayahSuSuSu/Android-DataBackup) є безкоштовним з
відкритим вихідним кодом для створення резервних копій на Android 8+.

## Root

Цей додаток потребує root, підтримує
[Magisk](https://github.com/topjohnwu/Magisk),
[KernelSU](https://github.com/tiann/KernelSU),
[APatch](https://github.com/bmax121/APatch).

## Встановити

Ви можете отримати **APK** з:
* **[Випуски](https://github.com/XayahSuSuSu/Android-DataBackup/releases)**
* **[F-Droid](https://f-droid.org/zh_Hans/packages/com.xayah.databackup.foss/)**
* **[IzzyOnDroid](https://apt.izzysoft.de/fdroid/index/apk/com.xayah.databackup)**

В наявності 6 опцій:
* **foss**: Збудовано без [Firebase](https://firebase.google.com/).
* **premium**: Збудовано з [Firebase](https://firebase.google.com/).
* **[arm64-v8a](https://developer.android.com/ndk/guides/abis#arm64-v8a)**: Це
  ABI є для 64-bit ARM CPUs, **що є першим вибором для більшості користувачів**.
* **[armeabi-v7a](https://developer.android.com/ndk/guides/abis#v7a)**: Це ABI є
  для 32-bit ARM CPUs. Воно включає Thumb-2 та Neon.
* **[x86](https://developer.android.com/ndk/guides/abis#x86)**: Це ABI є для
  CPUs підтримує інструкції які є загальновідомими як "x86", "i386", or "IA-32".
* **[x86_64](https://developer.android.com/ndk/guides/abis#86-64)**: Це ABI є
  для CPUs які підтримують набір інструкцій які є загальновідомими як "x86-64".
  **Це для тих користувачів які використовують емулятор ПК**
