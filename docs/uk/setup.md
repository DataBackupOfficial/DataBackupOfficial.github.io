## Root дозвіл
Спершу треба надати root дозвіл.

### * Magisk
Ви можете надати root дозвіл напряму в спливному вікні коли натиснете на картку
дозволу.

### * KernelSU/APatch
Спершу вам треба надати root доступ вручну в їх менеджері, потім перезавантажити
**DataBackup**.

### Кастомізований su файл
Для деяких пристроїв, `su` файл може бути не названий `su`, тому ви можете
відредагувати його вручну. Тільки запам'ятайте, що треба перезавантажити
**DataBackup** щоб воно набуло чинності.
